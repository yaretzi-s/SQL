# 订单重复数据排查 SQL 查询
## 需求描述
排查订单商品明细表中存在的**重复录入数据**。
定义：同一订单（`order_id`）+ 同一商品（`product_id`）对应**多条记录**即为脏数据。
需查询出重复的订单、商品及重复次数，并按重复次数降序排列。

---

---

## 标准 SQL 查询语句
```sql
-- 查找同一订单+同一商品的重复记录（数据清洗）
SELECT
    order_id,
    product_id,
    COUNT(*) AS repeat_count
FROM order_items
GROUP BY order_id, product_id
HAVING COUNT(*) > 1
ORDER BY repeat_count DESC;
```

---

## 输出字段说明
| 字段名        | 含义                                   |
|---------------|----------------------------------------|
| order_id      | 订单ID                                 |
| product_id    | 商品ID                                 |
| repeat_count  | 记录重复次数（出现行数）|

## 核心知识点
1. **分组统计**：`GROUP BY order_id, product_id` 按订单与商品组合分组
2. **重复判断**：`COUNT(*)` 统计组合出现的行数，即重复次数
3. **筛选重复**：`HAVING COUNT(*) > 1` 过滤出存在重复的脏数据
4. **业务用途**：用于**数据清洗**，定位错误数据，避免统计金额、发货数量出错
5. **排序展示**：按重复次数降序，优先处理严重重复数据
